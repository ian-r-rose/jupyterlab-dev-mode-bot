# jupyterlab-dev-mode-bot

> A GitHub App built with [Probot](https://github.com/probot/probot) that A GitHub bot generating binder links for JupyterLab PRs

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
