# binder-pr-bot
> A GitHub App built with [Probot](https://github.com/probot/probot) that A GitHub bot generating binder links for GitHub Pull Requests

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
